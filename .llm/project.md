# Project Charter

- Purpose: Build an MDX-friendly blog using Next.js 16 App Router and Tailwind CSS v4.
- Stack: Next.js 16, React 19, Tailwind CSS 4 via @tailwindcss/postcss, Zustand, motion.
- Dev scripts: pnpm dev, pnpm build, pnpm start, pnpm lint.
- Fonts/Theming: next/font Geist sans/mono; custom design tokens defined in globals CSS.
